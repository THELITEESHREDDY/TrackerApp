# Habit Tracker App

# Features
## Add Tasks/Habit
### Task
    1.Give start time and endtime in a day
    2.name
    3.description
    4.isCompleted
    5.accomplishment (how much it is done)

### Habit (Need to follow for 21 days)
    1.Name
    2.description
    3.start time and endtime (will be same throughtout habit building process)
    4.isCompleted
    5.Accomplishments

## Edit Habit/Task
    can change the for a day 

## Delete Habit/Task
    may effect the consistency rate for habit
